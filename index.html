🟢 Configurazione Capital.com Demo

Collega il tuo conto Capital.com Demo (denaro virtuale, prezzi reali). Autenticazione come da documentazione ufficiale (open-api.capital.com): API Key + email + password della API Key; la sessione usa i token CST / X-SECURITY-TOKEN.
📊 Asset disponibili con Capital.com: Forex · Materie Prime (CFD: Oro, Argento, Petrolio…) — denaro virtuale. SHORT consentito. Regolamentato UE: funziona dall'Italia.
Non hai la chiave? Crea un conto demo su capital.com, attiva la 2FA (obbligatoria per l'API), poi nella piattaforma: Impostazioni → Integrazioni API → Genera nuova chiave. Imposta una password della API Key (diversa da quella del conto) e copia la chiave.
